# gamelib

A collection of utility scripts for Unity.

## Installation

Add a dependency to `Packages/manifest.json`, like below:

``` json
{
    "dependencies": {
        "com.ebalzuweit.gamelib": "https://github.com/ebalzuweit/gamelib.git#upm"
    }
}
```
